# ac-conector-client

## Init your AC7 connection
```bash
$ npm install
$ ./node_modules/.bin/ACCManager
# Interface Access started, please open :
# http://localhost:4545
# Create a Connection named 'My Connection Name'
```

## Use it
Download all JSSP
```bash
$ node index.js
$ tree download
└───xtk_jssp
    ├───acx
    ├───crm
    ├───nms
    └───xtk
```
